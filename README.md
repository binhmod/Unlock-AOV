# Unlock-AOV
Unlock fps AOV
# Sử dụng
Dán lệnh này vào Termux, nhớ setup Termux trước nhé!

`git clone https://github.com/binhmod/Unlock-AOV.git && cd Unlock-AOV && sh unlock.sh`

• Hoặc đơn giản hơn:
Tải file unlock.sh về, khuyên là chạy trên Termux là nó tự tạo.
